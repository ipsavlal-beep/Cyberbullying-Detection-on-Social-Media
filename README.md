# Cyberbullying-Detection-on-Social-Media
Social media platforms have become an integral part of modern communication, enabling billions of users to connect, share, and interact online. However, this rapid growth has also given rise to a alarming increase in cyberbullying — a form of online harassment that involves the use of digital platforms to intimidate, humiliate, or threaten individuals. Cyberbullying has severe psychological consequences, particularly among adolescents and young adults, including anxiety, depression, and in extreme cases, self-harm.
Manual moderation of harmful content is neither scalable nor efficient given the enormous volume of posts generated daily. This has created a pressing need for automated detection systems capable of identifying cyberbullying in real time. Natural Language Processing (NLP) and Machine Learning (ML) offer powerful tools to address this challenge by enabling computers to understand and classify human language at scale.
This project develops a cyberbullying detection model trained on the Davidson et al. (2017) Twitter Hate Speech dataset, comprising approximately 24,000 real tweets. Using TF-IDF vectorization for feature extraction and three classification algorithms — Logistic Regression, Naive Bayes, and Linear SVM — the model automatically identifies whether a given social media post constitutes cyberbullying, contributing to safer and more responsible online environments.
