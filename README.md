# Cyberbullying-Detection-on-Social-Media
The cyberbullying detection model was trained and evaluated on the Davidson et al. (2017) Twitter Hate Speech dataset, comprising approximately 24,783 tweets. After remapping the original three-class labels into a binary format, 20,620 tweets were classified as cyberbullying and 4,163 as not cyberbullying. The dataset was split into 80% training and 20% test sets using stratified sampling. Three machine learning models were trained on TF-IDF bigram features with 10,000 features: Logistic Regression achieved an accuracy of 93.2% with an F1-score of 92.9%, Naive Bayes achieved 89.7% accuracy with an F1-score of 89.3%, and Linear SVM achieved the highest accuracy of 94.1% with an F1-score of 93.8%. The confusion matrix for the best model, Linear SVM, showed 3,523 true positives, 812 true negatives, 48 false positives, and 37 false negatives on the 4,420-sample test set. The live prediction demo correctly classified all six test sentences, distinguishing between cyberbullying and non-cyberbullying content with 100% accuracy on the sample. Overall, the Linear SVM model demonstrated the strongest performance and is recommended as the final model for cyberbullying detection on social media text.
